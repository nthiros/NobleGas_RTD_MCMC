#  thd = Trinidad Head, USA (41N, 124W, 120m) (flask only)

#  
#  See (http://www.esrl.noaa.gov/gmd/obop/) for station statistics and personnel.
#  
#  The last column of data is a six digit binary number that lists the measurement projects used in calculating
#  the combined global mean.  A number 1 indicates data from the corresponding project was uses.
#  The binary number is arranged from left to right: oldGC, RITS, otto, CATS, CCGG, MSD
#  
#  oldGC = Original flask program started in 1977 at NOAA baseline observatories, expanded to 7 sites.
#  RITS = In-situ program started in 1987 at 4 stations (NWR added in 1990).
#  OTTO = New flask analysis GC installed in 1995, currently 13 background sample sites.
#  CATS = New in-situ GCs installed from 1998-2000 at 5 stations.  Summit, Greenland added in 2007.
#  CCGG = Carbon Cycle Gas Group flask program, a subset of sampling stations co-located with the "OTTO" program.
#  MSD = Flasks analyzed on a mass selective detector.
#  
#  nan = not a number or no data, and space(s) is the delimiter.
#  
#  The hemispheric and global mean error estimates are calculated using 1-standard deviation errors assigned to
#  the individual programs plus a systematic errors due to smoothing and interpolating.
#  
#  The combined data set is a weighted average of several background NOAA/GML measurement
#  programs. All programs are on the same NOAA scale and every attempt is made to correct known biases
#  between measurement programs; however, there maybe small differences that are not understood and
#  characterized. When comparing measurements from site to site, be aware that not all sites are
#  composed of the same measurement programs. The hemispheric and global means in the combined data set
#  are our best measure of long-term trends and are used for international and national assessments for
#  background air measurements.
